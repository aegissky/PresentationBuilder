# PresentationBuilder — 로컬 LLM 설정

> 본 배포본은 **자체완결**. 외부 스킬 경로 없이 본 폴더의 `_GUIDE/SKILL.md`만으로 동작한다.

---

## §1. 스킬 우선순위

| 순위 | 위치 | 역할 |
|------|------|------|
| 1 (최우선) | `_GUIDE/SKILL.md` | 발표 생성 7-Phase 방법론 |
| 2 | `PROMPT-STANDARDS.md` | LLM 호출 프롬프트 표준 |
| 3 (선택) | 사용자가 자체 추가한 스킬 | 프로젝트별 확장 |

## §2. 프로젝트 진입 시 자동 흐름

LLM이 본 폴더(또는 본 폴더로 부트스트랩된 새 PPT 폴더)를 작업 디렉터리로 인식하면:

1. `_GUIDE/SKILL.md` 읽기
2. `pres-config.json` 존재 확인 (없으면 인터뷰 모드)
3. `_input/` 폴더 자료 탐색
4. 사용자 트리거 키워드 대기 — `발표 만들어줘`
5. SKILL.md Phase 0 → 7 순차 진행

## §3. 사용자 오버라이드

본 파일을 자유롭게 편집하여 다음을 추가 가능:

- 조직별 톤·문체 규칙
- 추가 발표 유형 매뉴얼
- 자체 스킬 등록 (`_GUIDE/` 하위에 SKILL.md 추가)

오버라이드 우선순위: 본 파일 > `_GUIDE/SKILL.md` > 기본 동작
